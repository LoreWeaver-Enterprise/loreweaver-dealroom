Real-time narrative runtime in action

Every player interaction costs the studio €0. Director runs entirely on the player's hardware. No cloud. No API fees. No data leaves the device.

Director generates NPC dialogue, manages mood and trust systems, drives dynamic quests, and adapts to player choices at runtime. First dialogue line appears in 1.5 seconds on an RTX 3060. The XSD schema validation layer ensures the LLM can never corrupt game state.

Cloud competitors achieve lower raw latency using large GPU clusters. Director trades slightly higher latency for zero per-session cost, complete data privacy, and offline capability. For RPGs, strategy, and adventure games, 1.5s TTFT is within acceptable UX range.

What Does Cloud AI Cost Per Game?

Consider a single-player RPG like The Witcher 3: 80 hours of gameplay, roughly 20 NPC interactions per hour. That is 1,600 cloud API calls per player, per playthrough.

DirectorCloud AI (at €0.02/interaction)
Interactions per playthrough1,6001,600
Cost per playthrough€0€32

A game that retails for €60 would spend €32 per player on cloud inference alone. That is more than half the retail price, before platform fees, before taxes, before any other cost of goods. With Director, the narrative AI cost per player is zero because inference runs on the player's own hardware.

Cloud AI narrative does not work economically for story-driven games. On-device inference is the only viable approach.
