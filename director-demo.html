Cloud competitors achieve lower raw latency using large GPU clusters. Director trades slightly higher latency for zero per-session cost, complete data privacy, and offline capability. For RPGs, strategy, and adventure games, 1.5s TTFT is within acceptable UX range.

What Does Cloud AI Cost Per Playthrough?

Consider a single-player RPG like The Witcher 3: 80 hours of gameplay, roughly 20 NPC interactions per hour. That is 1,600 cloud API calls per player, per playthrough.

ScenarioCost per InteractionAssumptionCost per Playthrough
Low€0.003Small model, minimal context, text only€4.80
Medium€0.02Mid-range model, full game state context€32
High€0.05Large model, full context + voice synthesis€80

Even the conservative estimate: €4.80 per player per playthrough. For a €20 indie game, that is 24% of the retail price going to inference costs alone. For a €60 AAA title, the medium estimate (€32) consumes more than half the price before platform fees, taxes, or any other cost of goods.
